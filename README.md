# CryptKeep

A zero-knowledge password manager. Your vault is encrypted locally with AES-256 before anything touches the server — the password never leaves your device.

Built with Flutter and Supabase.

**Features**

- AES-256-CBC encryption with PBKDF2-SHA256 key derivation
- Import from KeePass (.kdbx)
- Categories, search, and bulk delete
- Cross-platform: macOS, Windows, iOS, Android
